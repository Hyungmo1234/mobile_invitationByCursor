# TTF 폰트 적용 가이드

## 1단계: 폰트 파일 준비
1. 받은 TTF 파일을 `public/fonts/` 폴더에 넣어주세요
   - 예: `public/fonts/MyCustomFont.ttf`

## 2단계: layout.tsx 수정
`app/layout.tsx` 파일에서 주석 처리된 부분을 찾아서 실제 폰트 파일명으로 수정하세요:

```typescript
// 예시: 폰트 파일명이 "MyCustomFont.ttf"인 경우
const customFont = localFont({
  src: './fonts/MyCustomFont.ttf',
  variable: '--font-custom',
  display: 'swap',
})
```

## 3단계: HTML에 폰트 적용
`app/layout.tsx`의 `<html>` 태그에 폰트 변수를 추가하세요:

```typescript
<html lang="ko" className={`${playfairDisplay.variable} ${notoSansKR.variable} ${customFont.variable}`}>
```

## 4단계: CSS에서 사용
`app/globals.css` 또는 컴포넌트에서 사용:

```css
/* CSS 변수로 사용 */
.my-custom-text {
  font-family: var(--font-custom), sans-serif;
}
```

또는 Tailwind 클래스로 사용하려면 `tailwind.config.js`에 추가:

```javascript
fontFamily: {
  custom: ['var(--font-custom)', 'sans-serif'],
}
```

그리고 컴포넌트에서:
```tsx
<p className="font-custom">커스텀 폰트 텍스트</p>
```

## 여러 폰트 파일이 있는 경우
여러 weight나 style이 있다면:

```typescript
const customFont = localFont({
  src: [
    {
      path: './fonts/MyCustomFont-Regular.ttf',
      weight: '400',
      style: 'normal',
    },
    {
      path: './fonts/MyCustomFont-Bold.ttf',
      weight: '700',
      style: 'normal',
    },
  ],
  variable: '--font-custom',
  display: 'swap',
})
```
