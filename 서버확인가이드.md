# 서버 상태 확인 가이드

## 1. 서버가 실행 중인지 확인

### PowerShell에서 포트 확인
```powershell
netstat -ano | findstr :3000
```

**결과 해석:**
- `LISTENING`이 보이면 → 서버가 실행 중
- 아무것도 안 보이면 → 서버가 실행되지 않음

### 프로세스 확인
```powershell
tasklist | findstr node
```
Node.js 프로세스가 실행 중인지 확인

## 2. 서버 시작하기

### 프로젝트 폴더로 이동
```powershell
cd C:\Users\c694\mobile_invitation_byCursor
```

### 서버 실행
```powershell
npm run dev
```

**정상 실행 시 보이는 메시지:**
```
▲ Next.js 14.x.x
- Local:        http://localhost:3000
- Network:      http://218.146.11.31:3000
- Ready in 2.5s
```

## 3. 서버 중지하기

터미널에서:
```
Ctrl + C
```

## 4. 포트가 이미 사용 중일 때

### 포트를 사용하는 프로세스 찾기
```powershell
netstat -ano | findstr :3000
```
마지막 숫자(PID) 확인

### 프로세스 종료
```powershell
taskkill /F /PID [PID번호]
```
예: `taskkill /F /PID 18556`

## 5. 서버가 응답하지 않을 때

1. **터미널 확인**: 에러 메시지가 있는지 확인
2. **브라우저 콘솔**: F12 → Console 탭에서 에러 확인
3. **서버 재시작**: `Ctrl + C`로 중지 후 `npm run dev` 다시 실행

## 6. 네트워크 접속 확인

### 로컬 접속 테스트
브라우저에서: `http://localhost:3000`

### 외부 IP 접속 테스트
브라우저에서: `http://218.146.11.31:3000`

**참고**: `package.json`의 `dev` 스크립트에 `-H 0.0.0.0`이 있어야 외부 접속이 가능합니다.
